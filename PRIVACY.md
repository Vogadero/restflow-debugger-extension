# RestFlow 隐私政策

## 数据收集声明

RestFlow 浏览器插件尊重用户隐私，以下是关于数据处理的完整说明：

### 🛡️ 我们收集的数据

**无个人数据收集**
- 本插件不收集、不传输、不存储任何个人身份信息
- 所有 API 请求数据仅存储在用户本地浏览器中
- 不会将任何数据发送到远程服务器

### 💾 本地存储的数据

以下数据仅存储在您的浏览器本地：
- API 请求历史记录
- 环境变量配置
- 插件设置偏好
- 工作空间数据

### 🌐 网络权限说明

插件需要网络权限用于：
- 发送用户手动触发的 API 请求
- 捕获浏览器 Network 面板的请求信息
- 不用于任何数据收集或跟踪目的

### 🔒 数据安全

- 所有数据使用浏览器原生 Storage API 存储
- 数据不会跨设备同步（除非用户主动启用浏览器同步）
- 卸载插件时所有数据自动清除
